# 🐇 Документация Интерфейса "Rabbit Hole UI v1.0"

## 1. Концепция дизайна
**Стиль:** Кибер-минимализм / Текстовый терминал.
**Основная идея:** Интерфейс как "окно в бездну". Минимум графики, максимум динамических эффектов, создаваемых кодом (SVG/Canvas). Это обнуляет затраты на API генерации изображений.

## 2. Цветовая палитра (HEX)
- **Фон (Void):** `#050505` (почти черный)
- **Акцент (Matrix Cyan):** `#00ffcc` (основной текст и свечение)
- **Предупреждения (Warning Gold):** `#f96` (для глубоких уровней)
- **Текст (Soft Gray):** `#cccccc` (для логов и второстепенной инфы)

## 3. Ключевые визуальные компоненты

### A. Динамический фон (The Vortex)
Вместо статичной картинки используется HTML5 Canvas.
- **Эффект:** Концентрические круги или фракталы, которые пульсируют в такт генерации текста.
- **Логика:** Чем глубже уровень (Depth), тем быстрее вращение и выше частота пульсации.

### B. Счетчик глубины (Depth Meter)
- **Расположение:** Верхний левый угол.
- **Вид:** `ГЛУБИНА: 12.4m` (число плавно меняется при каждом ответе).
- **Эффект:** Легкое мерцание (Glow), имитирующее старый ЭЛТ-монитор.

### C. Окно логов (The Scroll)
- **Механика:** Текст появляется с эффектом "печатающей машинки".
- **Стиль:** Моноширинный шрифт (`Courier New` или `JetBrains Mono`).

## 4. Техническая реализация (Frontend)

```html
<!-- Базовая структура для вставки в проект -->

  <canvas id="vortex-bg"></canvas>
  <header>
    <div id="depth-counter">УРОВЕНЬ: 0</div>
  </header>
  <main id="chat-window">
    <!-- Сюда ИИ выводит свои метафоры -->
  </main>
  <footer>
    <input type="text" id="user-input" placeholder="Прыгнуть глубже..." />
  </footer>
</div>
```

## 5. Промпт для доработки визуалов (для LLM)
Если ты захочешь, чтобы ИИ изменил этот интерфейс, используй этот промпт:
> "Создай минималистичный веб-интерфейс на HTML/CSS/JS в стиле темного терминала. Используй Canvas для создания фонового эффекта пульсирующих кругов. Основной цвет #00ffcc. Добавь поле ввода без рамки и окно чата, где текст появляется постепенно. Весь код должен быть в одном файле."

## 6. Преимущества для РФ-рынка
1. **0 рублей себестоимость:** Не нужны Midjourney/DALL-E.
2. **Мгновенная загрузка:** Весь интерфейс весит меньше 10 Кб.
3. **Атмосфера:** Выглядит как секретный софт или хакерский терминал, что идеально ложится в концепцию "Норы".
