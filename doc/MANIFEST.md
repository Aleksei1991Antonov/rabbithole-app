## 🎯 1. Концепция продукта
Приложение предлагает формат **«интеллектуального путешествия»** — управляемого диалога, который помогает снизить уровень стресса и трансформировать негативный эмоциональный опыт через авторское повествование.

- **Ключевые ценности:** Скорость работы, минимализм интерфейса, визуальная наглядность.
- **USP (Уникальность):** Авторский гипнотический стиль в сочетании с адаптивным сценарием.
- **Платформа:** Экосистема мессенджер MAX.

## 🛠 2. Технологический стек
- **Frontend:** React, TypeScript, Vite.
- **Styling:** Tailwind CSS + Lucide Icons.
- **Payment System:** ЮMoney
- **Open Сode:** GitHub Pages
